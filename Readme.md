# ASCII Picture Maker

---

__ASCII Picture Maker__ - Program and open-source  for creating  ASCII art from PNG, JPEG, and TIFF. The software code is written in C#. ASCII-art can be in txt and jpg format

---

## The ability to set up

  1. image scale
  2. image gamma
  3. font
